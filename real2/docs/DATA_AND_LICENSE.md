# 데이터 출처 및 라이선스

## 사용 데이터

- 지역: 서울
- 유형: 축제·공연·행사
- 항목 수: 201건
- 제공 기관: 한국관광공사
- 서비스: 국문 관광정보 서비스 TourAPI 4.0
- 원본 API: https://www.data.go.kr/data/15101578/openapi.do
- 라이선스: 공공누리 제3유형(출처 표시 + 변경 금지)

## 저장 원칙

`netlify/functions/data/seoul-festivals.json`은 제공받은 `서울_축제공연행사.json`을 이름만 영문 경로로 옮긴 파일이며, 내용은 바이트 단위로 동일하게 유지합니다.

원본 데이터의 필드를 수정하지 않고 Function에서 읽기 전용으로 사용합니다. 화면 표시용 날짜 형식과 검색 결과 객체는 실행 시 별도로 생성합니다.

## 사이트 출처 표기

최소한 다음 내용을 사이트에서 보이도록 유지합니다.

```text
이 서비스는 한국관광공사 Tour API(TourAPI 4.0)의 데이터를 활용하였습니다.
출처: 한국관광공사
라이선스: 공공누리 제3유형
```

## 데이터 한계

- 고정 JSON이므로 실시간 데이터가 아닙니다.
- 빈 문자열 또는 null인 필드는 정보 없음으로 처리합니다.
- `mapx`는 경도, `mapy`는 위도이며 원본은 문자열입니다.
- 축제 일정·장소·시간·프로그램·요금은 항목별로 누락될 수 있습니다.
- JSON에 없는 리뷰·평점·주차·교통·날씨는 추측하지 않습니다.
